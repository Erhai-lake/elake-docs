阴影

作用:给元素设置阴影效果

属性名:box-shadow

属性值: X轴偏移量 Y轴偏移量 模糊半径 扩散半径 颜色 内外阴影

::: warning

  • X轴偏移量和Y轴偏移量为必须
  • 默认是外阴影,内阴影需要添加inset :::
