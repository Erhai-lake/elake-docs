# 功能单元

> [!warning]
>
> 以下代码我知道你还没学,我也还没写
>
> 现在是整体学习,不要求你能看懂以下代码!

```java
public static void printHelloWorld() {
    System.out.println("Hello World");
}
```

其中`printHelloWorld`就是这个功能的名字,也叫**方法名**,如果以后要使用这个功能,就根据功能的名字找到这个功能运行

例子:

```java
public static void main(String[] args) {
    printHelloWorld();
}

public static void printHelloWorld() {
    System.out.println("Hello World");
}
```
